Данный пример демонстрирует работу с объекторм Range. 
Пользователь вводит слово, или несколько букв в текстовом поле, с целью найти в тексте документа.
При этом на странице вызывается функция findme(cValue), где в параметре cValue передается введенное пользователем строка.

Если JavaScript скрипт найдет, указанную пользователем строку, тогда, вместо сообщения "введите слово для поиска" появится сообщение, сколько найдено вхождений слова, а также, появится кнопка "Перейти", которая при нажатии выделяет найденый фрагмент текста. При следующем нажатии выделяется следующий найденный текст.
